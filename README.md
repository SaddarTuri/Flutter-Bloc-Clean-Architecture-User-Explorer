# Flutter-Bloc-Clean-Architecture-User-Explorer
A modern Flutter application built using BLoC state management and Clean Architecture principles. This project demonstrates scalable app structure, API handling, theme switching, and smooth navigation.

# Features
1. BLoC State Management
Efficient and reactive state handling using Flutter BLoC

2. Clean Architecture
Separation of concerns (Presentation, Domain, Data layers)

3. API Integration
Fetch and display users from a remote API

4. Navigation Menu
Smooth and structured app navigation

5. Dark / Light Theme
Toggle between day and night modes
⚡ Optimized Performance
Clean and maintainable codebase
